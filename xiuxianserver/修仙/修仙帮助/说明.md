# 修仙帮助组件说明

修仙帮助组件负责网页帮助站、帮助图片和固定导航入口。

## 命令

```text
帮助
修仙帮助
指南
```

## 规则

- `帮助` 返回网页帮助站地址，并引导使用 `修仙帮助` 或 `指南`。
- `修仙帮助` 返回 `修仙帮助/help.png` 图片。
- `指南` 返回关键二级组件按钮入口，只放固定命令，不放需要用户补参数的命令。
- 组件启动时递归读取 `修仙` 下所有 `.md` 文件并缓存；网页请求只读取内存缓存，避免频繁扫描文件造成服务器压力。
- 首页 `主要命令` 从各二级组件说明的 `## 命令` 章节生成；每个组件说明应在 `## 命令` 下放一个 `text` 代码块，代码块内每行一条命令。
- 命令行支持精细格式：`命令｜说明｜参数说明`；旧的一行一命令仍然兼容，网页会自动标记 `无需参数`、`固定选项` 或 `需要参数`。
- 首页搜索会按命令、组件文档和设定文档生成独立结果区，不再直接摊开所有栏目过滤。
- 移动端滚动后会收起搜索框，只保留紧凑顶栏；点击 `搜` 可重新展开搜索。
- 文档卡片和详情页会展示 Markdown 来源文件和最后更新时间。
- 修改 Markdown 后需要重启项目，帮助页缓存才会刷新。
- 网页入口为 `/xiuxian/help`，公开域名从 `.env` 的项目级 `PROJECT_DOMAIN` 读取。
- 首页按组件分组展示文章流，详情页展示完整 Markdown 和右侧目录。
- `后台接口` 当前不提供帮助接口，后续 web 后台再扩展。
