# 保险箱组件说明

保险箱用于冻结保存重要物品，避免被出售、回收、上架、使用、升级或附魔误操作。

## 命令

```text
保险箱 / 查看保险箱
存入保险箱 物品名 数量
存保险箱 物品名 数量 / 放入保险箱 物品名 数量
存入保险箱 宝石名 等级 数量
存入保险箱 武器#ID
取出保险箱 物品名 数量
取保险箱 物品名 数量
取出保险箱 宝石名 等级 数量
取出保险箱 武器#ID
```

武器写法支持 `武器#ID`、`#ID`、`武器 ID`；不支持只写裸数字，避免和物品数量混淆。

## 规则

- 默认容量 18 格。
- 背包物品、纳戒物品、宝石和备用武器都可以存入。
- 可叠堆物品按来源、物品、等级合并，同一种只占 1 格。
- 武器按实例 ID 存入，每把武器占 1 格。
- 已装备武器和最后一把武器不能存入保险箱。
- 保险箱内物品从原库存移走，取出前不能被使用、出售、回收、上架、升级或附魔。
- 取出后回到原来的库存：背包物品回背包，纳戒物品和宝石回纳戒，武器回武器库。
- 取出背包物品时仍会检查背包负重、格子和堆叠上限。

## 组件关联

- 上游库存来自背包、纳戒、宝石库存和武器库；存入后从原库存移走。
- 下游取出回到原库存，继续由背包使用、纳戒消耗、装备镶嵌、武器附魔、商场出售或二手市场上架。
- 商场、二手市场、武器、装备和纳戒在处理出售、上架、升级、附魔、开孔等动作时，都必须把保险箱视为冻结区，不能直接消费箱内物品。
- 保险箱不改变物品定义、武器属性、宝石等级和世界物资流向，只提供误操作保护。
