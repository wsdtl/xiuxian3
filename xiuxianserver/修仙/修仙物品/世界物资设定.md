# 世界物资设定

本文固定物品大类、用途边界、最终名称清单和落地口径；代码实现必须按本文执行。

## 一、总边界

修仙物品分为两套体系：

- 世界物资：世界流动、地点消耗、跑商搬运、战斗掉落和回收处理的物品。
- 角色成长道具：直接改变玩家、装备、武器或战斗能力的物品。

判断标准：

- 能被玩家直接使用、装备、镶嵌、附魔、升级、洗髓或改变战斗属性的，是角色成长道具。
- 不能直接让玩家变强，只参与经济、地点、补给、建设、回收或掉落流动的，是世界物资。

世界物资统一走六大类。武器、技能书、宝石、恢复药、开孔器、洗髓液、淬锋丹、铭刻之羽和原石不进入六大类。

### 古界遗迹来源

探险地点和太虚秘境统一视为古界遗迹的外显入口。玩家在探险、秘境和战斗中取得的世界物资、战利品，都应有古界遗迹气质：像是从旧灵田、废城、残阵、古兽巢、沉没宗门、断裂山门、荒废丹坊或遗迹战场中带出的东西，而不是普通路边杂货。

纯经济物品是例外。纯经济只表示本界各地区之间流通的地方交易物，不属于其他位面或古界遗迹产物；它们可以有修仙地域风味，但核心仍是地方商货、凭证、器物和特产，用来服务跑商差价和地区供需。

非纯经济世界物资的命名必须同时满足两点：

- 有修仙世界和古界遗迹的质感，能让玩家相信它来自探险、秘境或遗迹战场。
- 用途方向一眼能看懂：药路能解释恢复药供给，民生能解释凡城必备消耗，建设能解释城池和山门如何抗灾、防妖和发展，古物能解释神秘物质回收，战利品能解释特殊收购和战斗生态。

## 二、六大类世界物资

| 大类 | 核心用途 | 约束 |
| --- | --- | --- |
| 纯经济 | 11 个地区各 3 个特产，共 33 个，专门服务跑商复杂供需和地区差价 | 只放当前跑商入口特产，不收容额外杂物；不从探险和秘境掉落 |
| 药路 | 遗迹药材、丹坊余材、古炉火种，作为恢复药供给、顺药概率和商路补给效率的来源 | 不直接卖药，不让玩家通过商场买恢复药；需要对应现有恢复药 |
| 民生 | 古界遗迹中可被凡城吸收的大量基础物资，支撑城食、盐鲜、水净、衣被、燃安 | 可回收，不受普通价格曲线调控；名字要看得出用途 |
| 建设 | 古界建材、护城灵材、阵基异材和门面装饰物资 | 在城池/可回收地点回收后进入当地建设经验；必须能解释抗灾、防妖和发展 |
| 古物 | 含神秘物质的旧物，按含量档位分微蕴、中蕴、厚蕴，回收后进入当地城池蓄能 | 神秘物质只有数量多少之分；蓄能满后生成藏宝图拍卖，最终产出城池特色稀品武器 |
| 战利品 | 妖、魔、鬼、龙、兽、兵戈等遗迹战斗掉落物 | 主要走特殊收购和战斗生态，不参与普通跑商 |

### 地点与城池承接规则

城池不是额外命令面板，而是地点承接规则。玩家在哪里回收世界物资，物资就流向当前位置对应的回收城池；物品大类决定进入哪条城池账。

地点类型：

- 城池/可回收地点：只包含当前 11 个普通探险/跑商重合地点。玩家在这些地点回收普通世界物资时，当前位置就是承接城池。
- 太虚秘境：当前唯一秘境，是特殊战斗模式，文本和地点名带“秘境”后缀；秘境不作为回收城池。
- 虫洞：依附城池地点出现；特殊虫洞依附对应特殊物品回收地点出现。
- NPC 地点：系统自己使用的固定地点，存放在 `world_locations`，包括 11 个普通城池、太虚秘境、特殊收购点和系统回收点；宗门不能建立在 NPC 地点坐标上。
- 宗门：玩家建立的山门，不是 NPC 地点，也不是城池，不承接普通世界物资回收。
- 特殊收购点：镇妖司、伏魔殿、鬼市、龙渊阁、万兽盟、破军营只承接战利品战备，不承接药路、民生、建设、古物。
- 系统回收点：铸剑阁、琢玉楼、藏经阁只承接武器、宝石、技能书回收，不承接普通世界物资。
- 荒地：地图剩余没有城池、探险地、秘境、虫洞、特殊回收点或宗门的坐标，什么都没有。

城池范围：

- 只有 11 个普通探险/跑商重合地点承接世界状态：天枢城、青岚坊、赤霞港、玄铁岭、万药谷、云梦泽、流沙海市、寒霜关、雷泽城、碧潮岛、星陨墟。
- 这 11 个地点在“普通探险地点、跑商地点、城池状态承接点”三种身份上等价。
- 历史主题已经并入当前 11 个普通地点的地貌叙事和武器池，不再作为独立地点类别。

出售规则：

- 背包流通物统一走 `出售 物品名 数量` 或 `自动出售`；世界物资和战利品不设置额外玩家命令。
- 药路、民生、建设、古物由系统自动流入当前或最近的 11 个普通城池，并按物品大类自动分账。
- 战利品由系统自动流入对应特殊收购势力；纯经济继续走商场买卖。
- 药路物资在当地回收后，进入当地药路状态，影响顺药概率、补给效率和可顺药上限。
- 民生物资在当地回收后，进入当地民生底盘，并转化为天道恩赐，提高当地 `极·技能书` 概率。
- 建设物资在当地回收后，进入当地建设经验，推动当地城池等级成长。
- 古物在当地回收后，进入当地神秘物质蓄能，蓄能满后生成当地藏宝图拍卖。
- 战利品不走普通城池账，仍按实际出售流向对应特殊收购势力。
- 纯经济不走回收吸收，仍走跑商买卖差价。

展示规则：

- 不单独新增城池状态面板。
- 城池等级、建设进度、药路状态、民生底盘、神秘物质蓄能、藏宝图状态等信息，依托地图功能和玩家到达地点后的地点展示逐步呈现。
- 城池升级、藏宝图生成、药路补给明显变化、战备虫洞牵引等世界变化，可以进入系统通知、修仙早报或地点文本。

懒结算规则：

- 世界物资体系统一使用懒结算，不为药路、民生、建设、古物、藏宝图和战备另开常驻定时任务。
- 触发懒结算的入口：导航到地点、位置、地图、出售、自动出售、商场出售、虫洞/虫洞状态、虫洞奖励、修仙早报。
- 懒结算只补算“上次结算时间到当前时间”之间该地点或该势力应自然变化的部分。
- 宗门大会这类自然周结算仍走宗门模块自己的周期任务，不并入世界物资懒结算。

交易奖励：

- 药路、民生、建设、古物回收本质上仍是物品交易流动，玩家都应获得原石回报。
- 原石价格按大类价值、稀有度和当前地点需求计算；建设和古物的主要价值仍沉淀在城池发展，不应让原石收益压过跑商主收益。
- 回收行为可以被称号系统被动记录，用来体现玩家近期倾向，例如偏爱搬建材、倒腾古物、薅药路或补民生，但不新增主动称号命令。

回收价格基准：

| 大类 | 基础价格方向 | 调整规则 |
| --- | --- | --- |
| 药路 | 按 `base_price * 品质系数` 折算 | 状态值和原石都按基础价、品质和 `medicine_material_role` 计算，不按药名或小类名查表 |
| 民生 | 按 `base_price` 与品质档折算 | 不受普通价格曲线调控；`world_subtype_key` 只决定流入城食、盐鲜、水净、衣被、燃安哪条民生账 |
| 建设 | 按 `base_price` 与品质档折算建设经验 | 原石只是辛苦费；主要价值进入建设经验，满级后溢出经验丢弃 |
| 古物 | 按 `base_price / 115 * 品质系数` 折算神秘蓄能 | 不让古物变成普通倒卖货；主要价值沉淀为城池藏宝图蓄能 |
| 战利品 | 走特殊收购报价 | 不进普通城池账，战备蓄能按战利品基础价和品质折算，不按战利品展示小类名判断 |

规则字段：

- `effect.world_category` 和 `effect.world_subtype` 只作为当前皮肤展示名。
- `effect.world_category_key` 是世界物资分流规则键：`trade`、`medicine`、`life`、`build`、`relic`、`loot`。
- `effect.world_subtype_key` 是细分规则键：药路恢复组、民生五账、建设小类、古物含量档、战利品小类和纯经济产地槽位都用它承接。
- `medicine_material_role` 只用于药路三类来源：`material`、`catalyst`、`fuel`。
- 代码禁止用当前中文物品名、小类名或品质显示名判断数值；需要差异时按稳定键、基础价、品质稳定键或随机权重计算。

地点展示落点：

- 不新增独立城池面板；到达地点、位置详情、地点文本承担展示。
- 普通玩家只看短文本：城池等级、影响半径、民生恩赐、药路状态、藏宝图是否出现。
- 进阶信息按需展示：建设经验进度、神秘物质蓄能、近期回收倾向、附近宗门受益情况。
- 文本必须短，不能把到达地点变成报表；只有非零、临界、异常或正在发生的状态才优先展示。

地点短文本候选：

- 城池：`城池 Lv.23，影响半径 23，民生恩赐偏盛。`
- 药路：`药路偏紧，药铺戒备中，顺药仍可碰运气。`
- 建设：`城建经验 12,400/18,500，护城梁正在补。`
- 古物：`神秘蓄能 8,940/9,000，藏宝图将出。`
- 战利品：`镇妖战备 620/1,500，近日虫洞可能牵引。`
- 纯经济：`此地特产偏热，商路差价还在滚。`

近期倾向称号候选：

- 补民生：`城食守望者`、`凡城补给官`、`灯火续命人`
- 搬建设：`垒城匠手`、`护城搬山客`、`阵基接线人`
- 倒古物：`遗图拾荒客`、`秘库经手人`、`旧宝摸金者`
- 薅药路：`看炉顺药人`、`药铺熟脸`、`丹火借道客`
- 出战利品：`战场收尸客`、`战备供货人`、`裂口开门匠`

称号规则：

- 称号继续被动刷新，不新增主动入口。
- 称号展示的是玩家最近一段时间的特殊机遇和动作倾向，不是永久勋章。
- 同一类倾向可以出现多个候选，系统按最近权重、次数和价值自动选最合适的一条。
- 称号规则变化时，只加到现有被动规则里，不单开成就系统。

状态表口径：

- `city_world_states`：每个承接城池一行，保存药路、民生、建设、古物的当前状态和上次懒结算时间。
- `world_material_records`：世界物资流水，保存玩家、地点、物品、数量、大类、小类、原石收入、进入哪条状态、创建时间；称号、人物志、早报和清理汇总都以它为准。
- `treasure_maps`：藏宝图状态，保存所属城池、拍卖/可拾取/已成交/已领取状态、坐标、当前价、最高出价者、出价次数、过期时间、生成时武器信息。
- `treasure_map_bids`：藏宝图出价流水，保存藏宝图、出价人、出价、是否当前最高、创建时间；用于返还锁定原石和审计拍卖。
- `war_prep_states`：特殊收购势力战备状态，保存势力、战备值、阈值、待牵引状态、上次牵引时间和上次懒结算时间；`loot_subtype` 保存 `yao/mo/gui/long/shou/bing` 稳定键，不保存展示小类名。
- `wormholes`：继续作为普通虫洞和战备虫洞共用事件表，必须保存事件类型、势力、词条、奖励倍率等结构化字段或 metadata。

清理周期：

- 当前状态表不清理：`city_world_states`、`war_prep_states`、未完成的 `treasure_maps`。
- 短流水保留 5 天：`treasure_map_bids`、已失效通知辅助、地点短文本辅助缓存。
- 长流水保留 7 天：`world_material_records`、已完成藏宝图、战备牵引记录。
- 清理前必须先把 `world_material_records` 汇总进 `player_lifetime_stats`，包括总回收次数、总数量、总原石收入，以及药路/民生/建设/古物分大类统计；汇总后才删除 7 天前明细。
- 未领取、未成交、可拾取的藏宝图不按 5/7 天直接删除，只按藏宝图自身规则重随机或归属。

## 三、纯经济

纯经济只包含当前 11 个跑商地区的 33 个特产。

纯经济的重点是地区供需关系，不是单个物品功能。它不从古界遗迹产出，不作为探险和秘境掉落，只在本界地方之间流转，承担跑商系统的基础商品职责。

### 纯经济闭环

定位：

- 纯经济物资只服务地区差价、商路流动和跑商收益记录。
- 它不参与探险掉落、恢复药补给、城市建设、神秘物质回收和特殊收购。
- 它是跑商赚差价的唯一基础商品，闭环必须干净单纯。

来源：

- 每个跑商/普通探险重合地点固定产出 3 个地方特产。
- 纯经济特产是本界地方商货，不是古界遗迹物资。
- 玩家只能在对应产地商场购买该地特产。

流动：

- 玩家在产地购买纯经济特产，带到其他地区出售。
- 售价由产地距离、地区需求、当日交易热度、全服普通跑商收益线和个人普通跑商收益线共同影响。
- 纯经济流动只改变交易记录、跑商热度和收益进度，不直接改变地点行业状态。

消耗：

- 玩家出售后，纯经济物资从背包消失，并转化为原石收入、手续费、交易记录和跑商奖励进度。
- 同地点短时间倒卖需要被限制，避免原地刷交易。
- 玩家当天普通跑商收益过高时，只降低利润倍率，不禁止继续购买或出售。

动态调控：

- 某地某货被大量购买，该地该货买价上升。
- 某地某货被大量出售，该地该货卖价下降。
- 全服活跃人数越高，普通跑商收益线越高，避免多人参与时收益过早被压死。
- 个人收益线用于防止单人长期一家独大；触线后应体现为利润倍率下降，而不是硬性额度拦截。

供需曲线：

- 每件纯经济特产记录本地买入热度、外地售出热度和个人今日收益。
- 买价倍率 = `1 + 本地买入热度系数`，范围 1.00-1.35。
- 卖价倍率 = `地区需求倍率 * 距离倍率 * 热度修正 * 收益线修正`。
- 地区需求倍率范围 0.85-1.45，用来体现 11 个地区之间的复杂供需。
- 距离倍率范围 1.00-1.35，距离越远越高，但不能盖过地区需求。
- 同一货物短时间大量倾销时，热度修正最低压到 0.65；恢复按小时缓慢回升。
- 全服收益线只调低利润倍率，不阻止交易；个人收益线更狠，最低可压到 0.35。
- 普通跑商最终利润倍率限制在 0.35-1.80，避免暴富和完全无收益。

现有代码对齐：

- 当前纯经济特产只按 `纯经济` 一个交易组处理，不混入药路、民生、建设、古物或战利品分类。
- 基础跑商价格已经按纯经济货物上调，现行纯经济基准倍率为 1.18。
- 同地点买入后 4 小时内不能原地出售。
- 单件普通跑商利润率上限约 45%，个人和全服收益压力只压利润倍率，最低保留 35%。
- 近 7 天活跃人数用于动态调整普通跑商软线，避免活跃人数增加后过早进入低利润。
- 当前地区需求表落在 11 个地点对 `纯经济` 的需求倍率；复杂供需只能维护这张需求表，不能把药路、民生、建设、古物或战利品并入普通跑商。

约束：

- 不进入特殊收购。

- 不直接提升角色、装备、武器或战斗能力。

文本方向：

- 纯经济文本要像地方货、凭证、票据、器物、商会炒作物。
- 可以带一点坑蒙拐骗和行商话术，但核心是倒买倒卖。
- 文本不强调神秘古界来源，也不写成修仙者炼宝材料。

探险冲突处理：

- 普通探险掉落不读取 `trade_goods` 里的当前位置纯经济特产。
- 普通探险的地点物资掉落改为读取当前位置主题世界物资池，池子只包含药路、民生、建设、古物和战利品。
- 太虚秘境不掉纯经济；秘境背包物资优先走古物、战利品和少量高阶建设物资。
- 跑商特产只通过商场购买进入背包，不能被探险、虫洞或首领直接掉落。

## 四、药路

药路不直接给玩家增加复杂材料。它表现为世界供给能力：

- 当地药材、丹材、炉火越充足，商路接触时越容易出现恢复药补给或顺药事件。
- 玩家不能在商场直接购买恢复药。
- 跑商本身不直接发药；药路只解释恢复药为什么能进入世界流动。
- 药路物资来自古界遗迹里的旧灵田、荒废药圃、残破丹坊和熄灭古炉，名字要能对应现有恢复药来源。

### 药路闭环

定位：

- 药路是恢复药补给生态，不是买药系统，也不是炼药材料系统。
- 药路不让玩家长期囤复杂材料，不新增主动炼药命令。
- 药路解释的是：为什么跑商接触地方势力后，修仙者能顺手弄到恢复药。

来源：

- 药路物资来自古界遗迹里的旧灵田、废药圃、残破丹坊和熄灭古炉。
- 每种恢复药对应 3 个来源物资，分别承担药性根基、成药辅料、开炉燃料三类语义。
- 药材决定有没有药性根基，丹材决定能不能成药，燃料决定丹坊或药铺能不能稳定开炉。

流动：

- 药路物资不直接作为复杂材料发给玩家。
- 药路物资进入地点行业状态，影响当地恢复药补给效率、可顺药库存上限、顺药事件概率和恢复药类型倾向。
- 跑商经过相关地点时，商路接触会把药材、丹材、燃料的流通表现为当地药铺、丹坊、药号的供给变化。

消耗：

- 跑商结算时，系统根据当前地点药路状态随机触发商路顺药。
- 触发后玩家获得少量恢复药，当地药路库存或补给热度被消耗。
- 顺药不是主收益，也不是玩家主动命令；它只是跑商结算的随机附带事件。
- 顺药文本表现为修仙者借护送、看炉、验药、压价、讲情面等方式顺手薅药。

动态调控：

- 全服恢复药消耗越高，药路补给需求越强。
- 某地药路来源充足时，顺药概率提高，顺药上限提高，对应恢复药更容易出现。
- 某地药路被频繁薅取后，顺药概率下降，药铺戒备提高，补给需要通过持续商路流动慢慢恢复。
- 药路调控的是恢复药世界流动性，不是让玩家用原石直接买药。

顺药曲线：

- 每个地点保存药路值、药铺戒备、可顺药库存和近 24 小时恢复药消耗压力。
- 药路值按药材、丹材、燃料三条来源分别累计，单条软上限 3000；超过软上限后继续计入，但收益按阶梯衰减。
- 药材、丹材、燃料同时充足时，顺药事件更稳定；缺一类时仍可能顺药，但对应恢复药权重和数量下降。
- 基础顺药概率 4%；药路值越高，最高提高到 18%。
- 全服恢复药消耗压力高时，额外提高 0-6%，表示地方药铺被迫开炉补货。
- 药铺戒备会降低顺药概率，最低把最终概率压到基础值的一半。
- 单次顺药数量以 1-3 个为主，药路很高且消耗压力高时才出现 4-5 个；如果当地可顺药库存不足，就按剩余库存给，库存为 0 时不触发。
- 顺药类型按当前地点药路来源倾向决定；对应药材、丹材、燃料越齐，对应恢复药权重越高。
- 每次顺药后，当地药路值和可顺药库存下降，药铺戒备上升；戒备按小时阶梯回落。
- 顺药是跑商结算随机附带事件，不新增主动命令，也不能用原石购买恢复药。

可顺药库存：

- 可顺药库存不是商店库存，玩家不能买，只能在跑商结算时被随机顺走。
- 当地药材、丹材、燃料越完整，系统按小时把药路值转化为可顺药库存。
- 推荐转化：每小时每种恢复药最多补 1 个；药路高、全服消耗高时最多补 2 个。
- 单地单种恢复药库存软上限 12；药路极盛时最高 20。
- 顺药触发后扣除对应恢复药库存；库存不足就是短缺，不自动凭空补齐。
- 短缺不是 bug。玩家不回收药路物资、不跑通商路，当地就应该缺药，只在文本中提示“药铺空柜、炉火不旺、掌柜嘴硬”。

药铺戒备：

- 每次顺药后，当地戒备 +8 到 +15，按顺药数量和恢复药稀缺度增加。
- 戒备建议 0-100；每小时自然下降 6，若当地药路三类来源都充足，每小时额外下降 2。
- 戒备 0-30：药铺松散，正常顺药。
- 戒备 31-70：药铺警觉，顺药概率明显下降。
- 戒备 71-100：药铺关门装死，顺药概率压到基础值附近。

约束：

- 玩家不能在商场直接购买恢复药。
- 药路物资不进入普通跑商特产，不承担跑商差价商品职责。
- 药路物资不作为玩家背包里的长期复杂材料玩法。
- 药路只影响地点行业状态和跑商顺药结果。
- 药路不直接提升角色、装备、武器或战斗能力。

文本方向：

- 药路文本要有修仙者强势、地方药铺吃瘪、但又不完全明抢的味道。
- 文本可以写成看炉火、验药性、护商路、压药价、讲人情、借名头。
- 文本重点是“坑蒙拐骗、顺手牵药、强势接触”，不是正经买卖。

背包提示：

- 玩家对药路物资发送 `使用` 时，提示去 11 个承接城池使用 `出售 物品名 数量` 或 `自动出售`，不要提示商场出售或特殊收购命令。
- 药路物资的背包说明要写清“回收后影响当地顺药，不是炼药材料，也不能直接吃”。

示例文本方向：

- 你替药铺掌柜看了眼炉火，顺手把账上那瓶回春露看没了。
- 丹坊说今日不开炉，你点了点伏火炭，掌柜立刻想起还有一瓶凝神露能孝敬。
- 药童刚想喊价，你已经把血契丹说成了护送商路的辛苦费。

## 五、民生

民生是凡城刚需经济品。它们不承担角色成长功能，也不是修仙者炼宝材料，而是百万凡人日常生存、城池运转和灾后稳定需要持续消耗的基础供应。

民生越高，代表当地凡城越稳、人烟越盛、香火越足，也就是天道恩赐越厚。天道恩赐不直接给玩家属性，而是提高当地掉落中出现 `极·技能书` 的概率。

民生约束：

- 可回收。
- 不受普通价格曲线调控。
- 不能和纯经济跑商特产混用。
- 来自古界遗迹，但用途必须一眼能看懂。
- 核心影响是当地天道恩赐，提升 `极·技能书` 的出现概率。

### 民生闭环

定位：

- 民生是城市状态底盘，不是跑商商品、炼制材料或角色成长道具。
- 民生解释的是：古界遗迹里带出的可用物资，如何被凡城吸收，支撑大量凡人和地方秩序。
- 民生方向固定为天道恩赐：凡城越稳，修士在当地越容易撞见没有副作用的上乘传承。

来源：

- 民生物资来自探险、太虚秘境和遗迹战斗中的古界残留。
- 来源语义应像旧灵田、沉没粮仓、废城水府、荒城库藏、残破军镇和避灾洞府中翻出的物资。
- 民生物资可以被玩家带回，但不设计成需要玩家长期囤积、配方加工或主动投喂城市的复杂材料。

流动：

- 民生物资不进入普通跑商买卖池，不使用纯经济地区差价逻辑。
- 玩家获得后，在城池/可回收地点通过现有回收动作流出背包。
- 民生流入地点后，记入当地民生底盘，并转化为当地天道恩赐。
- 跑商只间接触碰民生，当前阶段不把民生写成普通跑商差价货。

消耗：

- 民生消耗对象设定上是城池、坊市、关隘、海港和聚落，不是单个玩家。
- 城食、盐鲜、水净、衣被、燃安分别对应吃饭、盐鲜、净水、衣被和取暖安防。
- 当前阶段只固定分类、回收流向和天道恩赐影响，不新增玩家主动命令。

动态调控：

- 民生不受普通价格曲线调控，不能因为跑商热度被压成普通低价货。
- 民生越紧缺，当地天道恩赐越薄，`极·技能书` 概率接近基础值。
- 民生越充足，当地天道恩赐越厚，`极·技能书` 概率越高。
- 民生调控的是“城市是否稳、天道是否偏爱此地”，不是“玩家赚差价是否爽”。

天道恩赐与 `极·技能书`：

- `极·技能书` 是现有技能书的稀有变体，仍属于角色成长道具，不进入世界物资六大类。
- 普通技能书继续遵守武器设定里的既有规则：每本技能书必须有“主效果 + 清晰代价”，不能只给纯正收益。
- `极·技能书` 以同名普通技能书为模板，移除该模板里的负面代价。
- `极·技能书` 的正面效果在原技能书基础上提升 20%；精神消耗降低、蓄势加快这类本身是正收益的字段，也按正收益处理。
- 民生只提高 `极·技能书` 的出现概率，不直接改变已经掉落或已经附魔的普通技能书。
- `极·技能书` 只在会掉落技能书的玩法中替换出现，例如探险、太虚秘境、首领、普通虫洞和战备虫洞；不会通过商场直接购买。
- 掉落判定优先按当前战斗或掉落依附的地点读取当地民生；虫洞依附城池地点时读取所依附城池，特殊虫洞读取对应特殊回收地点。

`极·技能书` 生成曲线：

- 先按原玩法判定是否掉落技能书；只有已经命中技能书掉落时，才追加判定是否升级为 `极·技能书`。
- 基础升级概率为 0.2%，表示没有民生加持时也可能偶遇天赐传承。
- 民生有效值使用阶梯评分，不直接使用库存总量，避免大城池无限堆数值。
- 民生评分分为 0-10 阶；每阶提高 0.45% 升级概率。
- 最终概率 = `0.2% + 民生阶数 * 0.45% + 玩法修正`。
- 普通探险玩法修正 0%；太虚秘境 +0.4%；首领 +0.6%；普通虫洞 +0.8%；战备虫洞 +1.0%。
- 最终概率硬上限 6%，任何情况下不能超过 6%。
- 若某次掉落本身已经是稀品技能书，升级为 `极·技能书` 时只改变正负效果，不额外改变品质。
- `极·技能书` 名称使用 `极·原技能书名`，例如 `极·风刃书`。
- 附魔判重按原技能书模板计算，同一把武器不能同时附魔 `风刃书` 和 `极·风刃书`，避免用极版绕过同书限制。

数据形态：

- `极·技能书` 作为独立 `ring_item_defs` 物品落库，进入纳戒、查询、掉落和回收流程。
- `极·技能书` 的 effect 里保存 `enchant_id` 和 `base_enchant_id`；`enchant_id` 指向极版附魔效果，`base_enchant_id` 指向普通模板。
- 附魔判重、展示来源和文本说明按 `base_enchant_id` 识别同模板，避免普通版和极版叠同一本书。
- 技能书回收时，极版可以有更高估价，但仍走藏经阁技能书回收，不进入世界物资六大类。
- 修仙物品查询应显示：极版来源、基础模板、移除的代价、强化后的正面效果。

民生阶梯与衰减：

- 民生不是一次性充能槽，也不是简单库存上限；它是当地长期状态，按阶梯生效。
- 每个地点分别记录城食、盐鲜、水净、衣被、燃安五类民生值。
- 单类民生值越高，进入下一阶需要的数量越多，采用阶梯门槛而不是统一上限。
- 推荐阶梯门槛：100、240、430、680、1000、1400、1900、2500、3200、4000。
- 五类民生分别计算阶数后取平均，向下取整，得到当地民生阶数 0-10。
- 任一小类低于当前平均阶 2 阶以上时，当地民生阶数额外 -1，避免只堆单一物资刷恩赐。
- 衰减按阶梯发生：每日结算时，每类民生值按当前所在阶门槛扣除。
- 1-3 阶每日扣当前阶门槛的 6%。
- 4-6 阶每日扣当前阶门槛的 8%。
- 7-8 阶每日扣当前阶门槛的 10%。
- 9 阶每日扣当前阶门槛的 12%。
- 10 阶每日扣当前阶门槛的 15%，也就是每类每天扣 600。
- 低阶容易维持，高阶衰减明显；越接近高恩赐，维持成本越高。
- 民生值不会因衰减跌破 0；衰减只影响自然触发的掉落概率，不追回已经产生的奖励。

现有玩法关联：

- 探险和秘境：作为古界遗迹物资来源。
- 回收：作为当前阶段的主要去路，在哪里回收就记入哪里。
- 技能书掉落：当地民生越高，越容易把普通技能书掉落升级为 `极·技能书`。
- 药路顺药：只保留环境底盘关系，不作为当前主要影响。
- 跑商：只做间接背景，不参与普通买低卖高。
- 修仙界历史、早报和地点文本：可展示某地民生是否充足、天道恩赐是否浓厚。

预留边界：

- 民生不替代建设物资，不直接提供城池建设经验。
- 民生不直接发放技能书，只影响已有技能书掉落的升级概率。
- 民生可以接城市状态、早报、地点文本和药路环境，但不能抢走 `极·技能书` 这条核心影响。
- 民生不新增主动命令，不要求玩家专门投喂城市。

文本方向：

- 民生文本要有“大量、刚需、凡城离不开”的感觉。
- 物品可以带修仙气息，但玩家一眼要知道它能养活人、净水、御寒、照明、防疫或安城。
- 文本不写成高端炼宝材料，也不写成普通路边小杂货。

背包提示：

- 玩家对民生物资发送 `使用` 时，提示去 11 个承接城池使用 `出售 物品名 数量` 或 `自动出售`。
- 民生物资的背包说明要写清“回收后进入当地民生底盘，影响天道恩赐和 `极·技能书` 概率”。

| 小类 | 物资 |
| --- | --- |
| 城食 | 遗田灵粟、月井麦、地乳豆、青髓薯、玉壳谷 |

## 六、建设

建设物资用于当地回收后的城池建设经验和城池等级成长。

建设物资来自古界遗迹里的断城、残阵、旧山门和荒废洞府。它们不应只是普通砖瓦木石；基础建材可以存在，但必须混合护城灵材、阵基异材和门面装饰，能解释凡城为什么能抵御妖兽、灾厄和灵潮冲击。

建设约束：

- 不直接给玩家属性。
- 不直接作为武器、装备、宝石升级材料。
- 不进入普通跑商差价。
- 核心去路是在城池/可回收地点回收，转化为当地城池建设经验。
- 城池等级上限为 107 级，等级数值等于城池影响半径。
- 城池影响不是占地，不限制宗门建址，只为范围内宗门提供增益来源。

### 建设闭环

定位：

- 建设是全服长期公共工程，不是单个玩家短期升级线。
- 建设物资的核心价值不是卖原石，而是把城池从一点点亮到能影响整张地图。
- 城池等级越高，灵脉、商路、秩序、供养和护城能力辐射越远。

来源：

- 建设物资来自探险、太虚秘境和遗迹战斗中的古界废城、残阵、旧山门、古渠、镇妖设施和宗门残墟。
- 物资可以被玩家带回，但不设计复杂配方，不要求玩家攒齐套装。

去路：

- 玩家在城池/可回收地点回收建设物资，当前位置对应的城池获得建设经验。
- 回收后，物资从玩家背包移除，玩家获得原石，当前城池获得建设经验。
- 原石收益不应压过跑商；建设物资的主要价值必须体现在城池经验上。
- 不在荒野、宗门山门、秘境点回收建设物资，避免全服虚空投料。

城池等级：

- 每座普通城池独立保存等级和建设经验。
- 城池满级为 Lv.107。
- 城池每提升 1 级，影响半径增加 1 格。
- 城池满级后继续可以回收建设物资并给玩家原石，但建设经验不保存溢出；提高城池等级上限时，不追认旧溢出经验。
- 宗门山门坐标与城池坐标的直线距离小于等于城池等级时，宗门受该城池影响。
- 若宗门同时处于多个城池影响范围，以主影响城为主，其他城池只提供有限协同，避免增益叠爆。

升级经验：

- 城池升级经验按全服长期公共目标设计。
- 当前公式：升下一级所需建设经验 = `1200 * 当前等级^1.8 + 3000 * 当前等级`。
- 前期等级需要有明显体感，后期 80 级以后应成为服务器级长期目标。

物资经验：

| 建设小类 | 单件建设经验 | 定位 |
| --- | ---: | --- |
| 基础 | 400 | 普通建城底料，量大稳定 |
| 水火 | 550 | 城池运转、净渠、防火、引雷、锁潮 |
| 阵基 | 650 | 护城阵、聚灵、防灾、封妖核心 |
| 城防 | 750 | 防妖、防冲城、防战乱 |
| 华饰 | 1000 | 繁荣门面和高价值建设物 |

宗门影响：

- 建设不占宗门地盘，只提供宗门增益背景。
- 宗门是否受益，只看山门坐标是否落入城池影响半径。
- 距离越近，宗门增益越完整；越接近边缘，增益越弱。
- 城池影响的核心效果是提升宗门已有增益，不新造一套独立宗门属性。
- 主影响城最高可让宗门所有既有增益提升 50%；具体增益项仍由宗门模块自己决定。
- 建设物资本身不直接给个人属性，必须通过城池影响宗门，再由宗门模块结算。

宗门多城覆盖衰减：

- 一个宗门可同时落入多个城池影响范围，但只选 1 座主影响城。
- 主影响城优先取“覆盖强度”最高者；覆盖强度 = `城池等级 * 距离系数`。
- 距离系数 = `max(0, 1 - 宗门到城池距离 / (城池等级 + 1))`。
- 主影响城提供完整宗门增益提升，上限 50%。
- 第二座覆盖城只提供自身可提供增益的 30%。
- 第三座覆盖城只提供自身可提供增益的 15%。
- 第四座及以后只展示覆盖数量，不计入当前宗门增益；避免多城堆叠把宗门增益顶爆。
- 最终宗门增益提升 = `主影响城增益 + 协同增益`，整体硬上限 75%。
- 距离接近影响半径边缘时，距离系数很低，即使多城覆盖也只能提供微弱协同。

城池等级与宗门增益：

- 城池等级 1-20：最多提供 10% 宗门增益提升。
- 城池等级 21-50：最多提供 25% 宗门增益提升。
- 城池等级 51-80：最多提供 40% 宗门增益提升。
- 城池等级 81-107：最多提供 50% 宗门增益提升。
- 同等级段内按覆盖强度平滑增长，避免刚升级跨段时突然爆炸。

动态目标：

- Lv.1 到 Lv.20 是周边影响期，能让城池附近宗门很快有体感。
- Lv.20 到 Lv.50 是区域辐射期，城池之间开始出现明显交界和协同。
- Lv.50 到 Lv.107 是全图点亮期，属于长期公共工程。

文本方向：

- 建设文本要写成城池官署、工坊、阵师、匠头、守城军接收物资。
- 文本重点是“把遗迹里拆出来的好东西拿去修城”，不是普通卖废品。
- 可以有修仙者强势索价、城务捏鼻子认账的味道，但最终要表现为城池被实实在在垒高。

背包提示：

- 玩家对建设物资发送 `使用` 时，提示去 11 个承接城池使用 `出售 物品名 数量` 或 `自动出售`。
- 建设物资的背包说明要写清“回收后进入当地建设经验，影响城池等级和附近宗门增益”。

| 小类 | 物资 |
| --- | --- |
| 基础 | 古城砖、青罡石、玄灰泥、铁木梁、鳞纹瓦 |
| 城防 | 镇妖石、破邪木、惊妖铃、兽纹拒马、玄铁闸 |
| 水火 | 净渠玉、寒泉管、避火砂、引雷桩、锁潮箍 |
| 阵基 | 护城砖、聚灵砂、封妖灰、辟邪钉、镇宅锁 |
| 华饰 | 琉光檐、玉兽环、云纹壁、月庭灯、金纹坊 |

## 七、古物

古物是含神秘物质的世界物资。这里的神秘物质只按数量多少区分，不设计稳定、变化、媒介等复杂功能。

古物的核心去路不是普通出售，而是在当地回收后进入城池蓄能：

```text
古物在当地回收 -> 当地城池神秘物质蓄能 -> 蓄能达到上限 -> 生成藏宝图拍卖 -> 成交后获得该城池特色稀品武器
```

古物小类固定为：

| 小类 | 含义 |
| --- | --- |
| 微蕴 | 含少量神秘物质 |
| 中蕴 | 含中量神秘物质 |
| 厚蕴 | 含大量神秘物质 |

### 古物闭环

定位：

- 古物是城池藏宝图系统的蓄能材料，不是玩家直接使用的成长道具。
- 古物不直接给玩家神秘物质，不直接给属性，也不做复杂配方。
- 神秘物质只表示含量多少，所有差异最终只影响城池蓄能值。

来源：

- 古物来自探险、太虚秘境和遗迹战斗中的旧王庭、星门、白塔、沉船、古战场、旧神祭坛和失落文明残骸。
- 古物名字可以更像修仙者会争抢的旧宝，命名参考面不限于传统修仙法宝，也吸收异世界、奇幻 RPG、失落文明和远古遗珍的语感，但不直接照搬现成名物。

去路：

- 玩家在城池/可回收地点回收古物，当前位置对应的城池获得神秘物质蓄能。
- 回收后，物资从玩家背包移除，玩家获得原石，当前城池获得神秘物质蓄能。
- 古物价格可以按神秘物质含量和当地需求提高，但主要价值必须体现在城池蓄能和藏宝图拍卖上。
- 不在荒野、宗门山门、秘境点回收古物，避免神秘物质变成虚空公共池。

蓄能值：

| 古物小类 | 单件蓄能值 | 定位 |
| --- | ---: | --- |
| 微蕴 | 10 | 常见碎遗物，积少成多 |
| 中蕴 | 35 | 有明显神秘物质反应的遗珍 |
| 厚蕴 | 120 | 能直接推动城池出图的重宝 |

蓄能周期：

- 每座普通城池独立保存神秘物质蓄能。
- 一个蓄能槽就是一个周期，不按自然周清零。
- 蓄能未满时持续累计；达到上限时，本周期结束并生成 1 张该城池藏宝图拍卖。
- 城池同一时间最多存在 1 张未完成藏宝图；拍卖中、可拾取、宗主待领、已成交未领取都算未完成。期间蓄能可以继续累计，但不连续生成多张图。
- 生成藏宝图后，从城池蓄能中扣除本次上限；溢出蓄能保留到下一周期。

蓄能上限：

- 城池等级越高，能承载的神秘物质越多，出图也越难被低量古物刷爆。
- 当前公式：蓄能上限 = `5000 + 城池等级 * 180`。
- Lv.1 城池上限为 5180，Lv.50 城池上限为 14000，Lv.107 城池上限为 24260。

藏宝图拍卖：

- 藏宝图作为该城池的临时特殊特产展示，但不是普通跑商商品。
- 藏宝图挂牌持续 24 小时。
- 拍卖最多支持 10 次有效出价。
- 达到 10 次有效出价时直接封锤成交；未达到 10 次则 24 小时到期成交。
- 出价会锁定出价人的原石；被更高价覆盖后，上一位最高价者的原石解锁返还。
- 当前标价等于当前最高出价。
- 当前最高价者不能自己抬价。
- 有出价时必成交：24 小时到期后，藏宝图直接归当前最高价者；第 10 次有效出价也直接成交。
- 最高价相同时，以最后一个有效覆盖出价者为准；正常实现应要求新出价高于当前价，避免平价刷次数。
- 新出价必须高于当前价，最低加价取 `max(1000, 当前价 * 5%)`，避免用极低加价耗尽 10 次出价。
- 无人出价时，挂牌到期后藏宝图转为随机地点可拾取状态。
- 随机地点必须是地图范围内可到达坐标，优先落在生成城池影响范围内的荒地；若城池影响范围内没有有效荒地，再落到地图范围内其它荒地或玩家宗门坐标。
- NPC 地点是系统自己使用的固定地点，包含 11 个普通探险/跑商/城池状态承接点、特殊收购点、系统回收点、太虚秘境等秘境/特殊事件地点。
- 虫洞依附其它地点出现，不作为独立藏宝图落点；如果虫洞依附的坐标本身是 NPC 地点，也按 NPC 地点排除。
- 藏宝图不能掉到 NPC 地点，也不能掉到已有藏宝图坐标；只有荒地和玩家宗门坐标可以成为落点。
- 如果随机落点命中玩家宗门坐标，藏宝图不在地上等待拾取，而是直接归属该宗门宗主。
- 可拾取藏宝图不绑定玩家，谁先走到该地点并触发地点文本，谁获得藏宝图。
- 城池同一时间最多保留 1 张未完成藏宝图；拍卖中、可拾取、宗主待领、已成交未领取都会占用该城池出图位，直到领取完成才释放。

藏宝图拾取：

- 到达藏图坐标时，地点文本提示“此地有一张旧图被风沙压着”等短句，并给出领取按钮。
- 领取不需要额外竞价，不自动发到邮箱，也不跨地图远程领取。
- 藏图坐标被领取后清空，该城池才允许下一张藏宝图从蓄能中生成。
- 已成交藏宝图不会自动发放武器，买主需要发送 `领取藏宝图` 确认入手；这类未领取图也会进入通知栏提醒。
- 可拾取藏宝图保留 72 小时；超过时间仍没人拿，就重新随机一次地点，不返还城池蓄能。
- 宗门宗主获得的藏宝图走系统通知提醒，宗主需要用领取命令确认入手，不自动变成武器。
- 荒地藏宝图不写入 `world_locations`，不会成为 NPC 地点，只作为稀疏坐标事件；玩家到达该坐标时临时展示地点文本。

藏宝图武器：

- 成交后，赢家获得 1 把绑定该城池主题的稀品武器。
- 武器模板复用现有主题武器池规则：普通探险已按当前地点主题武器池掉落，藏宝图优先取该城池/同主题地点武器池。
- 若当前城池没有独立主题池，则按同地貌、同地区或同叙事主题的地点池兜底；最后才从全武器池抽取并改写为城池叙事。
- 出价只决定归属，不影响武器强度。
- 武器等级上限在藏宝图生成时固定，不随拍卖价格变化。
- 当前公式：武器等级上限 = `min(88, 68 + floor(城池等级 * 0.16) + 随机 0~3)`。

| 城池等级 | 藏宝图武器上限 |
| ---: | --- |
| Lv.1 | 68-71 |
| Lv.20 | 71-74 |
| Lv.50 | 76-79 |
| Lv.80 | 80-83 |
| Lv.107 | 85-88 |

约束：

- 藏宝图武器固定为稀品，但不是天然毕业武器。
- 藏宝图武器上限必须低于淬锋丹最终上限，保留宗门大会淬锋丹价值。
- 藏宝图不进入普通跑商买卖，不受普通价格曲线调控。
- 古物不直接提升角色、装备、武器或战斗能力，必须先经过城池蓄能和拍卖。

文本方向：

- 古物文本要有“城池鉴宝、秘库蓄能、遗图出世”的感觉。
- 拍卖文本要像修仙界公开竞价，不要写成普通商店售卖。
- 成交文本可以写成按图寻到城池范围内旧库、残阵、遗宫、沉船或秘藏，最终取出城池特色稀品武器。

| 小类 | 物资 |
| --- | --- |
| 微蕴 | 灰月碎币、旧王庭徽、星门断钥、白塔残页、雾海铜铃、失语镜砂、眠龙骨片、空舟铆钉 |
| 中蕴 | 灰烬圣杯、银环星图、界树枯种、无面王冠、夜航罗盘、封门魔典、逆潮沙漏、龙眠石碑 |
| 厚蕴 | 群星命轮、旧神冠冕、界门王钥、终焉圣杯、万象禁书、昼夜双镜、天灾方舟、原初界标 |

## 八、战利品

战利品是战斗生态物资，和普通地点经济分开。

战利品来自古界遗迹中的妖兽、魔影、鬼物、古龙残裔、荒兽和兵戈怨灵。名称要有战斗掉落质感，不能写成普通肉骨皮货，也不能混成跑商商品。

战利品的核心去路不是普通出售，而是通过现有特殊收购点进入战备蓄能：

```text
战斗掉落 -> 出售/自动出售 -> 对应势力获得战备蓄能 -> 战备满牵引异界虫洞 -> 玩家用现有虫洞命令挑战和领奖
```

### 战利品闭环

定位：

- 战利品是“杀出来的凭证”，不是跑商货、建设材料、古物或角色成长道具。
- 玩家出售战利品获得原石，这是当前直接收益。
- 对应势力收走战利品后，获得战备蓄能，这是战利品对世界玩法的长期影响。
- 战备蓄能必须绑定现有玩法落地，优先接入特殊收购、异界虫洞、早报和地点详情，不新增复杂命令入口。

来源：

- 普通探险和太虚秘境中的怪物战斗掉落战利品。
- 首领和虫洞奖励若产出世界物资，也优先从战利品池中按战斗类型补充。
- 战利品来源必须有战斗痕迹，不能从普通跑商、民生回收、建设回收或古物回收中产出。

去路：

- 玩家使用 `出售 物品名 数量` 或 `自动出售`。
- 每个特殊收购点只收对应战利品，并为对应势力增加战备蓄能。
- 自动出售跨多个特殊收购点时，按实际出售地点分别增加各自战备蓄能。
- 战利品出售收益继续使用特殊收购收益曲线；战备蓄能不额外提高出售价格，避免经济收益和世界事件叠爆。

蓄能值：

蓄能阈值：

- 每个特殊收购势力独立保存战备蓄能。
- 当前阈值 = `1800 + 活跃人数 * 240`，最低 1800，最高 6000。
- 战备未满时持续累计，不按自然日清零。
- 战备满后进入待牵引状态，优先在下一次对应势力收到战利品时结算。

虫洞牵引：

- 异界虫洞同一时间仍然只能存在 1 个开启中虫洞。
- 战备虫洞复用现有异界虫洞基础规则：普通持续 60 分钟，每日虫洞数量最低 2 次、基础 2 次、每 4 个活跃玩家 +1、最高 8 次。
- 战备虫洞和普通虫洞共用同一个每日次数池、同一个开启槽、同一套通知冷却和挑战冷却。
- 虫洞通知沿用现有 5 分钟冷却，挑战沿用现有 3 分钟冷却。
- 战备满不会强行突破虫洞唯一限制，而是生成待牵引资格。
- 若当前没有开启中的异界虫洞，且当天虫洞数量未达到上限，系统从待牵引队列中选择 1 个势力开启战备虫洞。
- 若已有虫洞开启，或当天虫洞数量已达上限，待牵引资格保留，战备不丢失。
- 牵引成功后，扣除本次阈值战备；溢出战备保留到下一轮。
- 战备虫洞继续使用现有 `虫洞`、`挑战虫洞`、`虫洞奖励` 命令流程，但记录类型、Boss 倾向、奖励倾向和文本都必须区别于普通虫洞。
- 战备虫洞不新增“战备虫洞”命令；玩家仍通过地图、通知、早报和现有虫洞入口感知。

虫洞事件数据：

- 普通虫洞和战备虫洞共用 `wormholes` 表。
- `wormholes` 需要记录事件类型：`normal` 或 `war_prep`。
- 战备虫洞需要额外记录：势力、战备名、战备词条、奖励倍率、定向奖励倾向、战备扣除值。
- 如果不新增多列，也必须把这些字段以结构化 metadata 写入 `result`，不能只拼在 Boss 名称里。
- `虫洞状态`、`虫洞排行`、`虫洞奖励`、系统通知和早报都从结构化字段读取战备信息，避免文本解析。

待牵引队列：

- 每个势力战备达到阈值后进入待牵引队列。
- 队列按达到阈值时间排序，先满先牵引。
- 若多个势力同一时间满，优先选择溢出战备更高的势力。
- 同一势力刚牵引过战备虫洞后，短时间内优先级降低，避免单一势力连续霸占虫洞窗口。
- 队列只决定下一个可开启的战备虫洞，不额外生成新命令或自动发奖。
- 队列短缺就短缺，不自动补战备；如果玩家不出售对应战利品，该势力就长期不牵引虫洞。
- 普通虫洞随机触发仍可照常发生；战备队列只在有开启资格时抢占下一次虫洞生成倾向，不关闭普通虫洞生态。

普通虫洞和战备虫洞：

| 类型 | 来源 | 特点 |
| --- | --- | --- |
| 普通虫洞 | 跑商、导航、普通交易随机触发 | Boss 名、异界法则、真实战斗模板分别随机，普通奖励倾向 |
| 战备虫洞 | 战利品蓄能满后牵引 | 势力定向 Boss，异界法则和真实战斗模板随机固化，定向奖励，带战备词条 |

战备词条：

战备词条是战备虫洞生成时随机附带的特殊状态。每个战备虫洞随机 1-2 个词条，同一词条不重复；词条会影响本次虫洞的 Boss、掉落或持续时间。

| 词条 | 效果方向 |
| --- | --- |
| 残巢已露 | 对应战利品掉落增加 |
| 旧门半开 | 洗髓液或特殊道具概率小幅增加 |
| 战痕回涌 | 武器掉落概率增加 |
| 余烬未冷 | 技能书掉落概率增加，命中技能书后仍按战备虫洞的 `极·技能书` 修正判定 |
| 群敌聚形 | Boss 更厚，奖励更高 |
| 裂口不稳 | 持续时间更短，奖励略高 |

战备虫洞数值：

- 战备虫洞基础难度按同等级普通虫洞提高 10%。
- Boss 血气提高 15%，攻击提高 8%，精神提高 8%；若出现“群敌聚形”，血气额外提高到 +35%。
- 战备虫洞基础奖励倍率为 1.20。
- “群敌聚形”奖励倍率额外 +0.15；“裂口不稳”奖励倍率额外 +0.10，但持续时间缩短 30%。
- “残巢已露”让对应战利品掉落权重提高 50%。
- “旧门半开”让洗髓液或特殊道具掉落权重提高 20%。
- “战痕回涌”让武器掉落权重提高 20%。
- “余烬未冷”让技能书掉落权重提高 20%；命中技能书后，`极·技能书` 升级仍使用战备虫洞总修正 +1.0%。
- 每个战备虫洞随机 1-2 个词条；同一词条不重复。
- 战备虫洞奖励倍率最终硬上限 1.50，避免和首领、宗门大会奖励定位互相踩踏。

文本示例方向：

- 普通虫洞：`异界虫洞撕开：千刃界游王 出现在 云梦泽。`
- 战备虫洞：`镇妖司连夜剖开三箩妖骨，终于从骨纹里摸到一条旧巢裂隙。战备虫洞撕开：百臂青妖王 出现在 镇妖司。`

展示落地：

- `特殊收购` 列表和特殊收购点详情应展示该势力当前战备状态，例如“镇妖战备 620/1500”。
- 战利品出售成功文本应追加本次战备增长和当前蓄能进度。
- 战利品自动出售应按各势力汇总展示战备增长。
- `虫洞` 状态面板应展示当前虫洞是普通虫洞还是战备虫洞；普通虫洞展示异界法则，战备虫洞展示势力、战备词条、异界法则和奖励倾向。
- `虫洞奖励` 文本应展示普通虫洞异界法则；战备虫洞还要展示定向奖励来源、词条和奖励倍率。
- `修仙早报` 可以展示最高战备、待牵引队列、即将牵引虫洞的势力和昨日被牵引的战事。
- 虫洞被战备牵引时，提示文本要写清楚是哪一方势力用战利品追踪到了异界裂口。

通知与早报：

- 系统消息队列只放全服/世界事实，默认展示队首 3 条；队首事件处理完、过期或消失后，排在后面的事件自然补上。队列项使用无框命令链接，点击后进入对应玩法或状态页。
- 战备虫洞开启、普通虫洞开启、岁时情劫开启、藏宝图拍卖将结束、散落藏宝图等世界动向进入系统消息队列，不混入个人通知。
- 个人通知栏只放需要当前玩家处理或会错过的事项：休息/探险可结束、奖励待领、对战请求、藏宝图可领取、宗主收到藏宝图、自己竞拍的藏宝图将结束、二手成交到账，以及新手礼包、银行结息、签到等低优先级日常；通知项使用无框命令链接，点击后自动发送对应命令。
- 药路短缺、建设进度、民生高低、神秘蓄能未满不进系统消息队列和个人通知栏，只放地点文本或早报。
- 修仙早报新增短栏目“城池风向”：最高建设城池、药路最紧城池、民生最盛城池、藏宝图动向、战备待牵引势力。
- 城池风向只展示 3-5 行，不做完整报表。

约束：

- 战利品不进入普通跑商商品池。
- 战利品不进入民生、建设、古物和药路状态。
- 战备蓄能不直接给个人奖励，也不直接给宗门增益。
- 战备只通过现有特殊收购动作增长，未出售到对应势力的战利品不计入战备。
- 战备满只生成待牵引资格，不额外生成新命令、新背包材料或自动发奖。
- 战备虫洞必须和普通虫洞共用同一开启槽，同一时间不能出现多个虫洞。

文本方向：

- 战利品文本要有“拿着尸骸证据去换赏”的粗粝感。
- 特殊收购点不是普通商人，而是各自势力在借玩家之手清理世界威胁。
- 战备满时的文本要像势力终于从一堆残骸里摸到了线索，然后顺藤摸瓜撕开虫洞。

## 九、角色成长道具

以下物品不进入六大类世界物资：

| 类型 | 原因 |
| --- | --- |
| 武器 | 独立实例，有等级、经验、上限、攻击、装备状态和附魔 |
| 技能书 | 消耗后改变武器附魔和战斗效果 |
| 宝石 | 镶嵌到装备，直接提供属性 |
| 恢复药 | 玩家主动使用，恢复血气或精神 |
| 开孔器 | 装备孔位成长材料 |
| 洗髓液 | 体质成长材料 |

## 十、代码落点

当前代码以本文六大类和物品名称为准：

1. `item_defs` 承接六大类世界物资和战利品定义。
2. 跑商特产只来自 11 个普通城池的纯经济物资。
3. 探险、太虚秘境、首领、虫洞和战斗掉落只从对应掉落池读取，不掉纯经济特产。
4. `出售` / `自动出售` 统一把药路、民生、建设、古物、战利品分流到城池状态或特殊收购。
5. `查看修仙物品` 展示世界物资大类、子类、主要来源和当前去路。
