# 二手市场组件说明

二手市场负责玩家之间的一包商品交易。每个卖家同一时间只能上架一包商品。

## 命令

```text
二手市场 / 小黄鱼
二手市场上架 名称 数量 总价 / 小黄鱼上架 名称 数量 总价
二手市场上架 宝石名称 等级 数量 总价
二手市场上架 武器#ID 总价
二手市场下架 / 小黄鱼下架
二手市场购买 卖家名称 / 二手市场购买 [CQ:at,qq=卖家ID] / 小黄鱼购买 卖家名称
```

## 规则

- 一个卖家同一时间只能上架一包商品。
- 可上架背包物品、纳戒物品、宝石和备用武器。
- 宝石按名称和等级区分；同名多等级时必须写清等级。
- 武器按实例 ID 区分，支持 `武器#12`、`#12`、`武器 12` 等写法。
- 武器上架会把武器临时托管到市场 owner，买家购买后再转移所有权。
- 已装备武器不能上架；玩家最后一把武器不能上架。
- 下架会把物品退回原卖家；背包物品退回时会重新检查背包空间。
- 购买时用卖家展示名称定位，也可以直接使用 CQ/at；WS 会把 CQ/at 转成内部标识。
- 买家支付总价，卖家收到扣除 5% 手续费后的源石。
- 买家不能购买自己的商品。
- 武器购买后，如果买家原本没有武器，会自动装备；否则作为备用武器。

## 扩展

- 市场手续费来自 `MARKET_FEE_RATE`。
- 新增可上架类型时，需要补齐上架扣库存、下架退库存、购买入库和展示名称四段逻辑。
- 武器展示使用实例差异化文本，避免同名武器难以区分。
