# 源库组件说明

源库负责源石存储和低息结算。源库里的源石不直接参与消费，消费前需要取出为随身源石。

## 命令

```text
获取源库 / 源库获取 / 源库
源库结息 / 结息源库
升级源库 / 源库升级
存入源石 数量 / 源石存入 数量
取出源石 数量 / 源石取出 数量
```

## 规则

- 源库不存在时会自动创建一星源库。
- 存入、取出、升级前都会先结息，避免利息和本金状态错开。
- 结息按距离上次结息的小时数计算，单次最多计算 24 小时。
- 每个源库等级有容量上限、小时利率、每日利息上限和升级费用。
- 每日利息按 04:00 业务日重置。
- 结出的利息会进入随身源石，不会继续自动存入源库。
- 存入数量超过容量时，只会存入剩余可用容量。
- 取出数量超过余额时，只会取出当前源库存量。
- 升级满级后会提示改用源石做装备、武器等成长。

## 扩展

- 调整源库等级：改 `BANK_LEVELS`。
- 调整最高等级：改 `BANK_MAX_LEVEL`。
- 结息事务逻辑在 `SourceVaultService._settle_conn()`。
