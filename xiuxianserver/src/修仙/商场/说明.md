# 商场组件说明

商场负责跑商、地点导航、特殊战利品出售和每日跑商奖励。它处理的是经济行动，不改变玩家本体战斗状态。

## 命令

```text
商场
商场列表
商场详情 地点名
商场行情 商品名
商场购买 商品名 数量
商场出售 商品名 数量
商场自动出售
商场推荐
跑商记录
跑商限制
跑商奖励
特殊收购
特殊出售 物品名 数量
特殊自动出售 / 自动出售战利品
导航 地点名 / 去 地点名 / 来 地点名
导航 x y / 去 x y / 来 x y
```

## 规则

- 跑商商品来自 `item_defs` 中的可交易特产，买到后进入背包，占格子和负重。
- 商场购买只能买当前位置出售的特产，不能远程购买。
- 普通跑商只能出售跑商商品；恢复类、宝石、技能书、武器不参与跑商。
- 同一地点买入的同一批货，4 小时内不能原地出售。
- 普通跑商出售价格会按最近买入价封顶，最高参考利润为买入价的 45%。
- `商场推荐` 只推荐当前能买、能装进背包、且有利润的商品路线。
- 普通买入手续费为 1%，普通出售手续费为 2%；装备宝石可能降低跑商手续费。
- 每日 04:00 重置跑商奖励；当天普通跑商出售达到数量和净利润要求后，可领取一次额外源石。
- 普通跑商净利润 = 普通卖出收入 - 卖出手续费 - 普通买入成本 - 买入手续费；特殊收购不计入普通跑商净利润。
- 特殊收购只收怪物战利品，必须到对应特殊地点出售。
- 特殊出售使用柔性降价曲线，当天卖得越多，后续收益倍率越低，最低保留 35%。
- `特殊自动出售` 会按可出售战利品自动导航到多个特殊收购地点并出售，最终停在最后一个出售地点。
- 导航可去商场地点、特殊收购地点、回收地点，也可按坐标寻找最近地点。
- 跑商、导航、特殊出售可能触发异界虫洞发现。
- 探险中仍可跑商，解释为行商化身在外行动；探险本体位置和预计算结果不被跑商改变。

## 扩展

- 新增跑商地点：改数据库初始化里的 `trade_locations`，并写入坐标和特产。
- 新增跑商商品：改 `item_defs`，确保商品可交易、基准价和重量完整。
- 调整价格特色：改 `TradeService.price()` 相关价格曲线和地点供需逻辑。
- 新增特殊收购：改 `special_buyers` 初始化，配置地点、坐标、收购物品和倍率。
- 新增回收地点：改 `recycle_locations` 初始化，武器、宝石、技能书回收入口会按地点类型判断。
