# 首领组件说明

首领组件负责“岁时情劫”玩法。每天都会出现一只首领；二十四节气和传统节日优先，普通日从“每日旧愿”首领池轮换。文案走中国风伤痛叙事，主要产出铭刻之羽、开孔器和洗髓液。

## 命令

```text
首领 / 岁时情劫
首领状态 / 岁时情劫状态
挑战首领 / 挑战岁时情劫
首领排行 / 岁时情劫排行
首领奖励 / 岁时情劫奖励
```

## 出现规则

- 每日 04:00 作为业务日刷新点。
- 第一次发送首领命令会懒加载生成当天岁时情劫。
- 当天命中二十四节气或传统节日时，优先使用对应节令首领。
- 普通日从代码里的 `DAILY_BOSS_DEFS` 每日旧愿首领池按日期稳定轮换。
- 春节、元宵、端午、七夕、中秋、重阳是高权重节日，铭刻之羽产出略高。
- 首领不需要移动到某个地点，出现当天可以直接挑战。

## 挑战规则

- 每人每个首领最多挑战 5 次。
- 每次挑战后冷却 30 分钟。
- 首领必须由玩家本体挑战，玩家必须空闲且血气大于 0。
- 探险中不能挑战首领；行商化身只能跑商，不能代替本体战斗。
- 首领血量按服务器玩家生态动态生成，不是固定后期专属怪。
- 首领被击破或次日 04:00 退去后，参与过的玩家可以领取奖励。
- 挑战默认返回简要战斗摘要，包含伤害、行动次数、双方技能和剩余血气精神。
- 发送 `战斗日志 开启` 后，挑战返回详细逐次出手日志；发送 `战斗日志 关闭` 后恢复简要摘要。
- 战败后玩家精神归零。

## 奖励

- 奖励按贡献、排名和首领权重计算。
- 可能获得源石、经验、恢复物、开孔器、洗髓液、宝石、技能书、武器和铭刻之羽。
- 珍贵物品均为随机掉落；贡献和排名只提高概率，不会让低贡献玩家失去基础掉落机会。
- 铭刻之羽只从岁时情劫产出，使用后即删除。

## 扩展

- 新增每日首领：扩展 `DAILY_BOSS_DEFS`。
- 新增节气首领：扩展 `SOLAR_TERM_RULES` 对应的 Boss 文案。
- 新增传统节日首领：扩展 `LUNAR_FESTIVAL_DATES` 和 Boss 文案。
- 调整掉率：改数据库表 `seasonal_boss_reward_rates`。
- 调整挑战次数和冷却：改 `SEASONAL_BOSS_MAX_CHALLENGES` 和 `SEASONAL_BOSS_CHALLENGE_COOLDOWN_MINUTES`。
- 调整首领战斗日志：改 `SeasonalBossService._challenge_log_block()`。
