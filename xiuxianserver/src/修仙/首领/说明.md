# 首领组件说明

首领组件负责“岁时情劫”玩法。首领按二十四节气和传统节日出现，文案走中国风伤痛叙事，主要产出铭刻之羽。

## 命令

```text
首领 / 岁时情劫
首领状态 / 状态首领
挑战首领 / 首领挑战
首领排行 / 排行首领
首领奖励 / 奖励首领
```

## 出现规则

- 每日 04:00 作为业务日刷新点。
- 当天命中二十四节气或传统节日时，第一次发送首领命令会懒加载生成当天岁时情劫。
- 春节、元宵、端午、七夕、中秋、重阳是高权重节日，铭刻之羽产出略高。
- 当天没有岁时情劫时，会提示下一次出现时间。
- 需要临时为当天补生成最近已过的首领时，可在 `src/修仙/首领/__init__.py` 里取消注释对应启动回调。
- 补生成只会在当天没有首领、且当天本身不是节气/节日时生效，会显示“岁时回响”来源，不覆盖原机制。
- 首领不需要移动到某个地点，出现当天可以直接挑战。

## 挑战规则

- 每人每个首领最多挑战 5 次。
- 每次挑战后冷却 30 分钟。
- 首领必须由玩家本体挑战，玩家必须空闲且血气大于 0。
- 探险中不能挑战首领；行商化身只能跑商，不能代替本体战斗。
- 首领血量按服务器玩家生态动态生成，不是固定后期专属怪。
- 首领被击破或次日 04:00 退去后，参与过的玩家可以领取奖励。
- 挑战返回详细战斗记录，包含逐次出手、技能、伤害、剩余血气和精神。
- 战败后玩家精神归零。

## 奖励

- 奖励按贡献、排名和首领权重计算。
- 可能获得源石、经验、恢复物、洗髓液、宝石、技能书、武器和铭刻之羽。
- 铭刻之羽只从岁时情劫产出，使用后即删除。

## 扩展

- 新增节气首领：扩展 `SOLAR_TERM_BOSSES`。
- 新增传统节日首领：扩展节日映射和 Boss 文案。
- 调整挑战次数和冷却：改 `SEASONAL_BOSS_MAX_CHALLENGES` 和 `SEASONAL_BOSS_CHALLENGE_COOLDOWN_MINUTES`。
- 调整首领战斗日志：改 `SeasonalBossService._challenge_log_block()`。
