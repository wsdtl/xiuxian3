# 异界虫洞组件说明

异界虫洞是跑商、导航和特殊出售时概率发现的限时世界 Boss。二级包只挂载 WS 命令，真实业务在根目录 `wormhole_service.py`，避免二级包互相导入。

## 命令

```text
虫洞
虫洞状态
挑战虫洞
虫洞排行
虫洞奖励
```

## 出现规则

- 商场购买、商场出售、导航、特殊出售、特殊自动出售等经济行动可能发现虫洞。
- 同一时间只会有一个开启中的虫洞。
- 虫洞持续 60 分钟，超时后关闭。
- 每日虫洞数量动态计算：最低 3 次，基础 3 次，每 3 个活跃玩家 +1，最高 12 次。
- 发现概率会受来源、活跃人数、今日已开启数量和每日上限影响。
- 虫洞通知有 5 分钟冷却，避免频繁刷屏。

## 挑战规则

- 挑战要求玩家已创建、处于空闲状态、血气大于 0。
- 玩家必须在虫洞出现地点才能挑战。
- 探险中不能挑战虫洞，会提示本体正在探险，行商化身不能代替本体战斗。
- 每次挑战后冷却 3 分钟。
- 虫洞 Boss 血量按当前玩家生态动态生成。
- 挑战默认返回简要战斗摘要，包含伤害、行动次数、双方技能和剩余血气精神。
- 发送 `战斗日志 开启` 后，挑战返回详细逐次出手日志；发送 `战斗日志 关闭` 后恢复简要摘要。
- 战败后精神归零。

## 奖励

- 虫洞结束后，参与过的玩家发送 `虫洞奖励` 领取。
- 奖励按贡献、排名和玩家等级计算。
- 可能获得源石、经验、恢复类物品、洗髓液、宝石、技能书和武器。
- 虫洞不产出开孔器，也不产出铭刻之羽。
