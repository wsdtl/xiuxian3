# 纳戒组件说明

纳戒负责不占负重的物品库存，主要保存恢复类、宝石、技能书、洗髓液、开孔器和铭刻相关消耗品。

## 命令

```text
纳戒
洗髓
```

## 规则

- 纳戒物品不占背包负重。
- 宝石在纳戒中按名称和等级分开显示。
- `纳戒` 只负责查看库存。
- 恢复类物品可以通过 `使用 物品名` 消耗，实际入口在背包组件。
- `洗髓` 消耗 1 个洗髓液并重骰体质。
- 洗髓液只通过活动 Boss 掉落。
- 宝石需要通过 `镶嵌`、`宝石升级`、`回收宝石` 消耗。
- 技能书需要通过 `附魔武器` 或 `回收技能书` 消耗。
- 开孔器需要通过 `开孔 装备位` 消耗。
- 铭刻之羽需要通过铭刻命令消耗；成功铭刻后直接删除。

## 扩展

- 新增纳戒物品：改 `equipment_item_defs` 初始化数据。
- 宝石分类必须写为 `宝石`，系统会自动使用 `gem_items` 按等级存储。
- 恢复类和洗髓等主动效果在 `item_effects.py` 中处理。
