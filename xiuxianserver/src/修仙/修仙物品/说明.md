# 修仙物品查看组件说明

修仙物品组件提供统一的物品定义查询入口。它查的是配置资料，不是玩家库存。

## 命令

```text
查看修仙物品 物品名
修仙物品查看 物品名
查看 物品名
```

## 可查看内容

- 背包物品：存放位置、分类、品级、重量、跑商状态、使用状态、基准价、效果和说明。
- 纳戒物品：分类、品级、目标类型、使用状态、效果和说明。
- 武器模板：武器类型、掉落地点、模板基础攻击和自带技能。
- 武器自带技能：威力、精神消耗、蓄势基准和说明。
- 技能书附魔效果：具体战斗效果和精神消耗变化。
- 体质资料：阶位、路线、体质值、稀有等级、效果和说明。

## 规则

- 玩家必须已创建用户才能查询。
- 这里只看定义，不看玩家身上有没有该物品。
- 武器实例有等级、攻击、附魔和自定义名，必须使用 `查看武器 武器ID`。
- 铭刻后的武器名和附魔名属于实例显示名，不会改变 `查看修仙物品` 查到的基础定义。
- 找不到物品时，会提示先从 `背包`、`纳戒`、`武器` 复制准确名称。

## 扩展

- 新增可查询定义时，优先在本服务补一个按名称读取和格式化方法。
- 展示效果统一使用 `format_effect()`，避免不同组件对同一效果写出不同文本。
